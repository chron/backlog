# Steph character set v1

Built-in generation using the shared contract in
`remaining-v1-prompt-contract.md`. Flat-key sources are processed with
`uv run tools/character-art/pipeline.py all`.

## Idle

References, in order: `odin-master-v1.png`, `steph/dubious.png`,
`steph/nervous.png`, `steph/neutral.png`, `steph/side-eye.png`.

Preserve Steph's oval face, long voluminous curly medium-brown hair, fair skin,
large light translucent glasses, slim build, expressive eyes, and subtle wry
smile. Depict a full-stack/mobile engineer who loves developer tooling and
automation: relaxed three-quarter stance, composed wry smile, one small precise
open-palm presenting gesture, other hand lightly at her waist. Outfit:
raspberry-red zip jacket open over a dark slate top and black trousers. Read
capable, thoughtful, practical, inventive, and lightly amused; never anxious,
timid, manic, or corporate.
